Comment ça marche

Ce panneau reproduit exactement le rendu du Grimoire dans Cantrip (theme, filtres, onglets de niveau). Clique sur un sort pour le modifier, ou sur le bouton + pour en ajouter un au niveau affiche.

Tes modifications sont sauvegardees automatiquement dans ce navigateur. Pour transmettre le resultat a Claude Code :

1. Clique sur "Exporter le fichier" une fois tes sorts ajoutes/modifies.
2. Donne le fichier telecharge a Claude Code avec l'instruction de mettre a jour SPELLBOOK et DENEOR_SPELLBOOK dans index.html, puis de commit/push.
3. Pour continuer l'edition plus tard ou sur ton autre PC, utilise "Importer un fichier" avec ce meme export (ou un export precedent).
