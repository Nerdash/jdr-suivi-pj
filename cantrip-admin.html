Token GitHub

Cree un Personal Access Token GitHub de type fine-grained, limite au depot Nerdash/cantrip, avec uniquement la permission Contents : Read and write. Il est enregistre uniquement dans le stockage local de ce navigateur et n'est jamais envoye ailleurs qu'a l'API GitHub (api.github.com).
